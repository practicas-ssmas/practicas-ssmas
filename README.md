# 🚀 Perfil de Desarrollo - Alumno en Prácticas (DAW)

Bienvenido/a al perfil de GitHub del estudiante en prácticas de **Desarrollo de Aplicaciones Web (DAW)** integrado en el equipo técnico de **[SSMÁS](https://ssmas.es/)**.

Este espacio está destinado a recopilar el progreso, el código y el aprendizaje generado durante el periodo de Formación en Centros de Trabajo (FCT).

---

## 🏢 Sobre las Prácticas en SSMÁS

El programa de prácticas de DAW en SSMÁS está diseñado para transformar los conocimientos teóricos del ciclo formativo en competencias profesionales reales dentro del sector tecnológico.

### 🎯 Objetivos Principales:
* **Desarrollo de Proyectos Reales:** Participación activa en el diseño, desarrollo y mantenimiento de aplicaciones web internas o de clientes.
* **Flujo de Trabajo Profesional:** Integración en el equipo técnico utilizando metodologías de desarrollo, control de versiones riguroso y resolución de incidencias (*bug fixing*).
* **Calidad de Código:** Adopción de estándares de clean code, refactorización y optimización de bases de datos.
